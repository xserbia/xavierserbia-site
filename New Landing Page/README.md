# Fórmula Dinero — Prototipos y modelo de precios

Mockups navegables y especificación del modelo de membresía de **Fórmula Dinero (FD)**.

## Contenido

| Archivo | Qué es |
|---|---|
| `index.html` | Página de inicio que enlaza a los prototipos |
| `fd-radiografia-flow.html` | Embudo completo: inicio → datos → 14 preguntas → resultado → plan personalizado |
| `fd-pricing-page.html` | Página de precios de las tres opciones |
| `FD-pricing-spec.md` | Especificación completa del modelo (niveles, FD IA, coaching, precios) |

## Cómo verlo

- **En vivo (GitHub Pages):** una vez activado Pages, abre `https://<usuario>.github.io/<repo>/`
- **Local:** descarga el repo y abre `index.html` en cualquier navegador

## El modelo, en una línea

Un *tripwire* de una sola vez (Radiografía, $19) → núcleo recurrente (**Estabilidad**, para Supervivencia/Estabilidad) → premium recurrente (**Crecimiento**). El Estadio define qué recibes; el compromiso define cuánto pagas.
