FAQ

Respuestas a las preguntas más comunes antes de comprar el Plan Revisión.

¿Listo para empezar?
¿Qué exactamente recibo al final del Plan Revisión? +
Un Discovery Map (tu mapa financiero inicial), una lectura clara de tu situación (liquidez, deudas, estructura), tus riesgos y prioridades principales y una recomendación de próximos pasos para decidir con intención.
¿Cuánto tiempo toma el proceso? +
El proceso completo toma entre 7-10 días desde tu compra, dependiendo de qué tan rápido completes tu información. Una vez tenemos el Discovery completo, se coordina la sesión guiada de interpretación y se entrega el resumen con próximos pasos.
¿Necesito tener todas mis cuentas perfectas para empezar? +
No. Empezamos con lo que tienes. El objetivo es ordenar y hacer visible tu realidad hoy. Si faltan datos, te indicamos exactamente qué completar.
¿Esto incluye asesoría de inversiones, impuestos o legal? +
No es asesoría de inversión, impuestos o legal. Es un proceso para organizar, entender y decidir. Si surge la necesidad de especialistas (CPA, abogado, etc.), se sugiere como próximo paso.
¿Qué pasa si compro y no tengo tiempo para completar el Discovery? +
Puedes completarlo cuando estés listo. El Discovery toma tiempo porque requiere recopilar información, así que el proceso avanza cuando tu data está completa. Tu acceso no expira.
¿Todo esto es confidencial? +
Sí. Tu información se usa únicamente para construir el Discovery Map y preparar la sesión. El acceso y el flujo se manejan desde tu cuenta de forma segura.
¿Desde qué monto de patrimonio/ingresos tiene sentido? +
Generalmente desde $5,000 USD en activos o ingresos mensuales de $2,000+. Si tienes menos, puede ser mejor empezar con recursos gratuitos hasta alcanzar ese nivel.
